# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.8.0] - 2026-01-01

### Añadido
- Nuevo módulo **DNI a PDF**: convierte imágenes de frente y dorso del DNI en un único archivo PDF
- Nuevo módulo **Sellar PDF**: permite agregar un sello con firma, fecha/hora automática y protege el documento contra edición
- Soporte para archivos de hasta 1GB en todos los módulos

### Mejorado
- Actualización de NativePHP de la versión 1.x a 2.x
- Sistema de actualizaciones optimizado: las alertas solo aparecen cuando hay una actualización disponible o al ejecutar una comprobación manual
